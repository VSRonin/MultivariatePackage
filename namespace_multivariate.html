double Multivariate::GammaFunction ( double  t)

Computes the Gamma function for positive reals.

Parameters
tThe point in which the function should be evaluated
Returns
The value of the gamma function

Evaluates the \( \Gamma \) function as \( \Gamma(t) = \int_0^\infty x^{t-1} e^{-x}\,{\rm d}x. \)

if t is 0 or negative the function returns 0.0

For more information, please refer to Wikipedia
